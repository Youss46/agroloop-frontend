My AgroLoopCI app is already built in Replit. I need you to configure it for production deployment on Railway with the custom domain agroloopci.ci. Do NOT touch any existing feature or logic — only apply the configuration changes below.

---

## 1. vite.config.js (inside /client folder)

Replace or update the file with:

```js
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
  build: {
    outDir: '../server/public',
    emptyOutDir: true,
  },
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:5000',
        changeOrigin: true,
      }
    }
  }
});
```

---

## 2. server/index.js

At the very bottom of the file, AFTER all API routes and middleware, add this block:

```js
import path from 'path';
import { fileURLToPath } from 'url';
const __filename = fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);

// Serve React frontend in production
if (process.env.NODE_ENV === 'production') {
  app.use(express.static(path.join(__dirname, 'public')));
  app.get('*', (req, res) => {
    res.sendFile(path.join(__dirname, 'public', 'index.html'));
  });
}
```

If the file already uses CommonJS (require/module.exports instead of import/export), use this version instead:

```js
const path = require('path');

if (process.env.NODE_ENV === 'production') {
  app.use(express.static(path.join(__dirname, 'public')));
  app.get('*', (req, res) => {
    res.sendFile(path.join(__dirname, 'public', 'index.html'));
  });
}
```

## 3. package.json (root level)

Make sure the root package.json has exactly these scripts. Do not remove existing scripts, only add or update these:

```json
"scripts": {
  "dev": "concurrently \"npm run server\" \"npm run client\"",
  "server": "node server/index.js",
  "client": "cd client && npm run dev",
  "build": "cd client && npm install && npm run build",
  "start": "NODE_ENV=production node server/index.js"
}
```

Also make sure these packages are in dependencies (add them if missing):

```json
"dependencies": {
  "concurrently": "^8.0.0",
  "dotenv": "^16.0.0"
}
```

Then run: `npm install`

Create a new file called railway.json at the root:

```json

  "build": {
    "builder": "NIXPACKS",
    "buildCommand": "npm run build"
  },
  "deploy": {
    "startCommand": "npm start",
    "restartPolicyType": "ON_FAILURE",
    "restartPolicyMaxRetries": 3
  }
}
```

## 7. Verify the PORT in server/index.js

Make sure the server listens on the PORT environment variable, not a hardcoded port:

```js
const PORT = process.env.PORT || 5000;
app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});
```

Run these commands in the Shell one by one and confirm each works:

